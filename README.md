Transcriptomic Profiling and Pathway-Based Target Identification in Parkinson's Disease

Dataset:
GSE99039 (GEO)

Methods:
- GEOquery
- limma
- fgsea
- ggplot2
- pheatmap

Results:
- Differentially expressed genes identified
- Significant pathways:
  * MYC Targets V2
  * E2F Targets
  * Oxidative Phosphorylation
  * MTORC1 Signalling
  * Unfolded protein response

- 450 pathway-associated candidate genes prioritized
